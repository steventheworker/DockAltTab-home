Introduction

Keep up with the present by by harnessing the power of Windows --esque app previews on the MacOS dock by supercharging your running copy of AltTab by today. And you don't even have to buy it. DockAltTab is a window management app for MacOS. It provides previews on the dock using the familiar AltTab interface (shortcut2 must be set to Cmd+`) to trigger another application's (AltTab) keyboard shortcuts (using Esc and Cmd+`) to show/hide an overlay of the open window previews / thumbnails (for all/current space(s)) for the app icon you're hovering on in the macOS dock.
